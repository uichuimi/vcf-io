Class Variant

  • java.lang.Object
    • vcf.Variant
  • All Implemented Interfaces:
    java.lang.Comparable<Variant> 

    public class Variant
extends java.lang.Object
implements java.lang.Comparable<Variant>
    Stores a vcf. chrom, position, ref, alt, filter and format are Strings. position is an integer, qual a double. Info is stored as a map of key==value. If value is null, key is treated as a flag.
