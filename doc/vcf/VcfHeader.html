        Adds a new complex header. If there is another header of the same type with the same ID, it is updated.
        Parameters:
        type - type of header. One of FILTER, INFO, FORMAT, contig, SAMPLE, PEDIGREE, ALT
        map - type=value with the content of the header
        Throws:
        VariantException - when header is not added due to malformed header: id est, required type no present or inconsistency in values
