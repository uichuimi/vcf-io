Class MapGenerator

    • vcf.io.MapGenerator

  • public class MapGenerator
extends java.lang.Object
    Returns a LinkedHashMap with the content of the line parsed. So "ID=AC,Number=A,Type=Integer" becomes a map. This class is convenient to parse almost any VCF header lines.
